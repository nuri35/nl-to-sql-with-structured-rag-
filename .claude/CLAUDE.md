# Project: nl-to-sql

NestJS-based natural language to SQL pipeline using LangChain.js.
Single-shot RAG over SQLite (Chinook dataset).

## Stack
- NestJS 11+ (Node 20+)
- TypeScript (strict mode)
- LangChain.js + OpenAI (gpt-4o-mini)
- SQLite via better-sqlite3
- Jest for testing

## Goals
- Production-quality code structure (DI, modules, custom exceptions)
- DataCamp tutorial pattern → NestJS-grade implementation
- Sergileme amaçlı, GitHub-ready
